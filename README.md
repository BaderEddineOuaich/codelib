# CodeLib
Programming books Flutter App using ITBOOK API 


## How to run
Run Commands: 
* flutter packages get
* flutter run --release


## API

Here is the [Latest ITBOOK Data API][ITBOOK].

[ITBOOK]: https://api.itbook.store/1.0
